This package contains flat rate shipping functionality for the getpaid
framework.

Configuration
=============


You've got 2 configuration options: `Flat Rate`_ and Percentage_


Flat Rate
---------

Will let you set `Flat Shipping Rate` that will be charged for shipping,
no matter how many items where ordered.


Percentage
----------

Calculate the shipping costs based on a percentage on the order costs.

`Flat Shipping Percentage` is applied to each item of the order,
and summed up.

Additionally you can set `Max Handling Charge` to limit the shipping costs
to this amount.
